## Console Log
Use: `cl`
Explanation: Will give you a `console.log('Test')` with the cursor ready for replacement of the test

## Flex Center
User: `flex center`
Explination: Makes the following css lines: `display: flex;` `justify-content: center;` `align-items: center;`
